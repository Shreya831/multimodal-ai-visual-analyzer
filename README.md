# Multimodal AI Visual Analyzer

An AI system that understands images and detects objects using deep learning.

This project combines **computer vision and transformer models** to analyze visual scenes and detect objects in images.

---

## 🚀 Features

- Detects objects in images
- Draws bounding boxes around detected objects
- Uses state-of-the-art transformer models
- Works on custom images

---

## 🧠 Model Used

Object detection is performed using **DETR (Detection Transformer)** from **Hugging Face**.

---

## 🛠️ Tech Stack

- Python  
- PyTorch  
- Transformers  
- Computer Vision  
- Matplotlib
